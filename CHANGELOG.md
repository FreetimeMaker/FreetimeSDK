# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-07-13

### Added
- **RevenueCat Integration**: Web-based billing for subscriptions and fiat payments.
- **New Crypto Providers**: Bitcoin (BTC), Ethereum (ETH), Litecoin (LTC), and Solana (SOL).
- **Serverless Architecture**: All providers now work without a custom backend.
- **F-Droid Compatibility**: Removed all proprietary SDK dependencies.
- **Global English Support**: Full documentation and UI strings in English.
